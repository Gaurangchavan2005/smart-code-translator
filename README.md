# 🚀 Smart Code Translator

An AI-powered full-stack web application that helps developers translate, analyze, optimize, and understand code across multiple programming languages.

## 🔥 Features

* 🔐 JWT Authentication + Google OAuth
* 💻 Code Translation (C++, Java, Python, etc.)
* 📊 Complexity Analysis
* ⚡ Code Optimization
* 📖 Code Explanation
* 🕘 History Tracking

## 🛠 Tech Stack

* Frontend: React + Vite
* Backend: Node.js + Express
* Database: MongoDB
* AI: Gemini / OpenAI

## 🚀 Run Locally

```bash
cd server
npm install
npm run dev

cd client
npm install
npm run dev
```
